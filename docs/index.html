プライバシーポリシー

最終更新日：2026年4月27日

本アプリ「huuwa」（以下「本アプリ」といいます）の運営者（以下「当方」といいます）は、本アプリにおけるユーザーの個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

1. 収集する情報

本アプリでは、以下の情報を収集します。

1.1 ユーザーから直接提供される情報

  • メールアドレス（アカウント作成時）
  • パスワード（暗号化して保存）
  • ユーザー名・表示名
  • プロフィール画像
  • 自己紹介文
  • 投稿コンテンツ（テキスト、画像、ボイスメッセージ）
  • ダイレクトメッセージの内容
  • フォロー／フォロワー情報
  • ブックマーク情報

1.2 自動的に収集される情報

  • デバイス情報（OSバージョン、デバイスタイプ）
  • アプリの利用状況
  • ログイン日時
  • IPアドレス
  • 通知トークン（プッシュ通知用）
  • 広告ID（IDFA / GAID）

2. 情報の利用目的

収集した情報は、以下の目的で利用します。

  1. 本アプリのサービス提供および運営
  2. ユーザーの認証およびアカウント管理
  3. ユーザー間のコミュニケーション機能の提供
  4. プッシュ通知の送信
  5. 広告の配信およびその効果測定
  6. 不正利用の防止およびセキュリティの確保
  7. 利用規約違反への対応
  8. お問い合わせへの回答
  9. サービスの改善および新機能の開発

3. 第三者への提供

当方は、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。

  1. ユーザーの同意がある場合
  2. 法令に基づく場合
  3. 人の生命、身体または財産の保護のために必要がある場合
  4. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合
  5. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合

4. 外部サービスの利用

本アプリは、以下の外部サービスを利用しています。これらのサービスにおける個人情報の取扱いは、各サービスのプライバシーポリシーに従います。

Google Firebase（Google LLC）

本アプリは、以下のFirebaseサービスを利用しています。

  • Firebase Authentication：ユーザー認証
  • Cloud Firestore：投稿・メッセージ等のデータ保存
  • Firebase Storage：画像・音声ファイルの保存
  • Firebase Cloud Messaging：プッシュ通知の配信

Firebaseのプライバシーポリシー：
https://firebase.google.com/support/privacy

Google AdMob（Google LLC）

本アプリは、Google AdMobを使用して広告を配信しています。AdMobは、ユーザーの関心に基づいた広告を表示するため、以下の情報を収集する場合があります。

  • 広告ID（IDFA / GAID）
  • デバイス情報
  • 大まかな位置情報（IPアドレスから推測）
  • 広告とのインタラクション

ユーザーは、デバイスの設定からパーソナライズ広告を制限できます。

  • iOSの場合：「設定」 > 「プライバシーとセキュリティ」 > 「Apple広告」
  • Androidの場合：「設定」 > 「Google」 > 「広告」

AdMobのプライバシーポリシー：
https://policies.google.com/technologies/ads

Googleのプライバシーポリシー（共通）

https://policies.google.com/privacy

5. ユーザーが投稿するコンテンツ

本アプリはソーシャルネットワーキングサービス（SNS）であり、ユーザーが投稿したコンテンツ（テキスト、画像、音声等）は、設定に応じて他のユーザーに公開されます。

5.1 公開範囲

  • 公開投稿：すべてのユーザーが閲覧可能
  • ダイレクトメッセージ：送信相手のみ閲覧可能

5.2 禁止事項

ユーザーは、以下のコンテンツを投稿してはなりません。

  • 違法または公序良俗に反する内容
  • 他者の権利を侵害する内容
  • 暴力的、性的、または差別的な内容
  • スパム、ハラスメント、なりすまし
  • 個人情報の無断公開

6. 不適切なコンテンツへの対応

当方は、ユーザーから通報された不適切なコンテンツに対し、24時間以内に確認・対応を行います。

6.1 通報機能

ユーザーは、不適切なコンテンツやユーザーをアプリ内の通報機能により報告できます。

6.2 ブロック機能

ユーザーは、特定のユーザーをブロックすることで、相互のやり取りを制限できます。

6.3 違反者への対応

利用規約に違反したユーザーに対し、当方は警告、コンテンツの削除、アカウントの一時停止または永久削除等の措置を講じることができます。

7. データの保管期間

ユーザーの個人情報は、ユーザーがアカウントを削除するまで保管されます。アカウント削除後は、法令で保管が義務付けられている場合を除き、合理的な期間内に削除されます。

8. データの安全管理

当方は、収集した個人情報の漏えい、滅失または毀損の防止その他の安全管理のために必要かつ適切な措置を講じます。本アプリのデータ通信には、業界標準のSSL/TLS暗号化を使用しています。

9. ユーザーの権利

ユーザーは、本アプリの設定画面から以下を行うことができます。

  1. プロフィール情報の編集・削除
  2. 投稿コンテンツの削除
  3. アカウントの削除
  4. 通知設定の変更
  5. ブロックリストの管理

10. 児童のプライバシー

本アプリは、16歳未満の方の利用を想定しておりません。16歳未満の方は、保護者の同意なく本アプリを利用しないでください。

11. 国際的なデータ転送

本アプリで収集された情報は、Google Firebaseのサーバー（米国を含む）に保存されます。Firebaseは、適切なデータ保護措置を講じています。

12. プライバシーポリシーの変更

当方は、必要に応じて本ポリシーを変更することがあります。重要な変更がある場合は、本アプリ内またはウェブサイト上で通知します。
